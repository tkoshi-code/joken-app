# プライバシーについて

さいこー雀は、対局記録・プレイヤー情報・予定・動画メモ・局面メモを利用者の端末内に保存します。これらの入力データを運営者のサーバーへ自動送信することはありません。

## 診断ログ

不具合調査のため、次の情報を端末内に最大100件保存します。

- アプリのバージョン
- ブラウザ、画面サイズ、言語、タイムゾーン、通信状態などの利用環境
- 登録データの件数
- 画面移動とボタン操作の種類
- JavaScript、ストレージ、通信、Service Workerなどのエラー情報

プレイヤー名、点数、予定名、メモ、動画URL、写真など、利用者が入力した内容は診断ログへ記録しません。

診断ログは自動送信されません。「設定 → 問い合わせ用診断情報」で利用者が内容を確認し、コピーまたはJSON書き出しを行った場合に限り、利用者自身が問い合わせへ添付できます。同じ画面からいつでも消去できます。

## アクセス解析

利用状況の把握と品質改善のため、Cloudflare Web Analyticsを利用しています。Web Analyticsは、ページの閲覧、参照元、ブラウザ・端末・国や地域などの利用環境、ページの表示性能をCloudflareへ送信して集計します。

Web AnalyticsのスクリプトはCookieを使用せず、さいこー雀が保存したlocalStorageやIndexedDBの対局記録・プレイヤー情報・予定・メモ・写真を読み取りません。個人を識別する目的では利用しません。

端末内の診断ログとJavaScriptエラーは自動送信されません。問い合わせ時に利用者自身が診断情報をコピーまたはJSON書き出しし、任意で添付する方式です。
