# 条件戦サポート

麻雀条件戦(トーナメント・ワンデー・リーグ戦)のポイント管理と条件計算を行うスマホ向けPWA。
仕様は [SPEC.md](SPEC.md) を参照。

## 使い方(スマホへの導入)

PWAのオフライン動作にはHTTPSでのホスティングが必要です。**GitHub Pages が最も簡単**:

1. GitHubにリポジトリを作り、このフォルダの中身(index.html 等)をpush
2. リポジトリの Settings → Pages → Branch: main を選択して保存
3. 発行されたURL(`https://<ユーザー名>.github.io/<リポジトリ名>/`)をスマホで開く
4. **ホーム画面に追加**
   - iPhone: Safariで開き、共有ボタン →「ホーム画面に追加」
   - Android: Chromeで開き、メニュー →「ホーム画面に追加」
5. 一度開けば以後は**オフラインでも動作**します

## ローカルでの動作確認(PC)

```
npx -y http-server joken-app -p 8321 -c-1
```

を実行して http://localhost:8321 を開く。

## データについて

- データは端末のブラウザ内(localStorage)にのみ保存されます
- 設定 → バックアップ(JSON書き出し)で定期的にバックアップしてください
- 機種変更時は書き出したJSONを新端末で「復元」してください
